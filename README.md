# dserp.ai - SAP FI/CO Close Acceleration Diagnostic

Professional SaaS diagnostic tool for SAP FI/CO professionals.

## Quick Start

1. Deploy to Vercel
2. Connect dserp.ai domain
3. Start sending link to prospects

## Tech Stack

- Next.js
- Tailwind CSS
- Vercel (hosting)

## Files

- `pages/index.js` - Main diagnostic tool
- `pages/_app.js` - App wrapper
- `pages/api/` - API endpoints
- `styles/globals.css` - Styling
